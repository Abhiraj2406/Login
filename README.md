# 🌫️ Glassmorphic Login Page

A modern and stylish **Glassmorphism Login UI** built using HTML & CSS.
This project focuses on creating a clean, aesthetic, and user-friendly login interface with a frosted glass effect.

---

## ✨ Features

* 🌫️ Glassmorphism (frosted glass UI)
* 🎯 Clean & modern design
* 🧊 Background blur using `backdrop-filter`
* 📱 Responsive layout
* 🖱️ Smooth input focus effects
* 🔒 Password field with visibility icon styling

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, backdrop-filter, box-shadow)
